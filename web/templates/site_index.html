VRFTools

Free utility tools for Mitsubishi VRF systems, built by Madison Innovations.

📊
MTDZ Viewer
Free · Browser-based

The problem: Mitsubishi Maintenance Tool captures detailed system data during startup and operation, but there is no way to view that data without having the software installed, and no graphing capability — making it difficult to present findings to engineers, facilities managers, or commissioning agents in a format they can work with. There is also no supported method to generate a startup report that can go into project closeout documents.

The solution: Upload your .MTDZ, .MTLZ, or .MTPZ file and get interactive graphs for every ODU and IDU, a system topology map, and a Word document report exportable from project closeout documents — all without any software installation.
⚙️
LEV Kit Configurator Coming Soon
Free · Mobile-friendly

The problem: LEV kit DIP switch settings are more complicated than standard FCUs and easy to get wrong in the field. There's also no good way to get those settings to a technician — the previous utility was an Excel file that required a PC, and techs were copying images from spreadsheets into submittal documents.

The solution: A mobile-friendly configuration wizard that walks through the required settings and outputs a clean, easy-to-read switch setting table that can be copied and shared on the spot. On desktop, a batch editor lets you configure multiple units at once and outputs settings in a format ready to drop into submittal documents or mechanical prints.
🗂️
Central Controller Config Tools
4 utilities · Free · Browser-based

The problem: Building a central controller config file requires manually entering the name and M-Net address for every group — data that already exists in the DSB file. Mistakes discovered mid-build often mean starting over, and common tasks like fixing group order, changing controller generations, or splitting a site database for field loading have no tooling support.

The solution: Upload your DSB .dsbx file and download fully built, loading-ready config files for each centralized system — no manual data entry. Additional tools handle group reordering, generation conversion, and splitting multi-controller databases into individually loadable files.
