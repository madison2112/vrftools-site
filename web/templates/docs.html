How to Use

DSBX to DAT
  1. Export a .dsbx file from Diamond System Builder.
  2. Upload the file on the DSBX to DAT page.
  3. Optionally expand the group list to review or drag-and-drop groups into a different order.
  4. Click Download for AE-C400 Tool or Download for AE-200 Tool depending on your controller hardware.
  5. Run the Initial Settings Tool and open the .dat file like normal.

If your project file contains multiple Centralized Systems a ZIP file will be returned with one .dat file per block.

Group Rearranger
  1. Upload an existing single-controller .dat file.
  2. Drag group cards to rearrange them. M-Net addresses travel with their group — only the slot number changes.
  3. Use Sort by Tag Name to automatically alphabetize IC groups, with Lossnay groups at the end.
  4. Click Download Modified DAT. The output preserves the original controller type.
Generation Change
  1. Upload a .dat file from any supported controller type (AE-200, EW-50, AE-C400A, EW-C50).
  2. The tool automatically detects the source type and converts to its opposite: AE-200 ↔ AE-C400A and EW-50 ↔ EW-C50.
  3. Click Download Converted DAT. All group data is preserved; only the controller schema changes.

If the file contains multiple controllers they will each be converted and returned as multiple .dat files in a ZIP file.

Database Decoupler
  1. Upload a multi-controller .dat database (e.g. an AE-200 master + EW-50 expansion unit combined file).
  2. The tool lists all detected controllers. Each gets its own output .dat file.
  3. Click Download ZIP to receive all individual files in a single archive.

If you upload a single-controller file an error will be shown — nothing to split.
