MTDZ Viewer

View, report on, and inspect Mitsubishi Maintenance Tool monitoring files (.MTDZ, .MTLZ, .MTPZ).

About These Tools

The MTDZ Viewer opens maintenance tool monitoring files directly in the browser. Upload your file below and select a tool to graph log data, generate a diagnostic report, inspect system topology with port connections, or download extracted raw CSV data — no software installation required.

All processing happens on the server. Uploaded files are deleted nightly at 1:00 AM Pacific Standard Time. Maximum file size: 50 MB. See the How to Use guide for detailed instructions.
