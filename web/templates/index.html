Central Controller Config Tools

Web-based utilities for Mitsubishi Electric AE-200 and AE-C400A controller configuration files.

→
DSBX to DAT
Convert a Design Build project file (.dsbx) to a controller configuration file (.dat) ready to load into ISToolAEC or ISToolAE.
.dsbx → .dat
⇅
Group Rearranger
Upload an existing .dat file and drag-and-drop groups into a new order before downloading the modified file.
.dat → .dat
⇆
Generation Change
Convert a configuration file between AE-200 and AE-C400A tool versions (and their paired EW controller types) automatically.
AE-200 ⇄ AE-C400A
⌫
Database Decoupler
Split a multi-controller .dat database into individual files — one per controller — packaged as a ZIP download.
multi .dat → .zip
About These Tools

All processing happens on the server. Uploaded files are automatically deleted after one hour. Maximum file size: 5 MB.

For detailed usage instructions and file format documentation, see the How to Use guide.
