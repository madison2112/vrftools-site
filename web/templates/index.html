Central Controller Config Tools

Web-based utilities for Mitsubishi Electric AE-200 and AE-C400A controller configuration files.

About These Tools

The central controller config file required to load group and address data into an AE-200 or AE-C400A already exists in your DSB file — it just isn't in the right format. The DSBX to DAT converter closes that gap, turning a .dsbx file into one loading-ready config file per centralized system with no manual data entry. The remaining tools handle the next most common pain points: reordering groups after the fact, converting between controller generations, and splitting a site database into individual controller files that can actually be loaded when the IT backbone isn't complete yet.

All processing happens on the server. Uploaded files are automatically deleted after one hour. Maximum file size: 5 MB. See the How to Use guide for detailed instructions.
