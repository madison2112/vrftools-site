How to Use

Upload a .dsbx file on the Central Controller Config Tools page, then click DSBX to DAT to open it here. Groups are auto-populated from the DSB data. You can double-click any group tag name to edit it inline, and click the controller name above the grid to rename it. Both are saved to the .dat and JSON exports. Drag cards to reorder groups before downloading.

1. Review & Optionally Rearrange Groups
2. Download

Choose the target tool version. If your project has multiple controller blocks, a ZIP file will be returned.
